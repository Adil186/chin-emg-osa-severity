# Canonical 41-feature schema

The submitted scripts mixed a 40-method toolbox list with a 41-column manuscript representation. The model schema used here resolves the ambiguity as:

- 37 scalar Jx-EMGT descriptors; plus
- four coefficients returned by the fourth-order autoregressive (`ar`) descriptor.

`MeanValueOfSquareRoot` (`msr`) and `LogCoefficientOfVariation` (`lcov`) are not part of the final 41 classifier columns defined by the published MRMR figure/model schema.
