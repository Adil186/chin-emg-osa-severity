# Data access and privacy

## ACPN

The ACPN polysomnography dataset and ACPN-derived feature matrices are not included. The data contain identifiable clinical information and are subject to ethics approval and data-protection restrictions. Only users authorized by the originating institution may run ACPN workflows.

A public manifest must never contain real clinical identifiers. The example manifests use synthetic identifiers only.

## UCDDB

UCDDB is obtained separately from its public host. This repository includes code and manifest templates but does not redistribute recordings or annotations.

## Prohibited repository content

Do not commit EDF files, MAT files containing clinical outputs, spreadsheets with subject-level data, real subject manifests, psychiatric-comorbidity labels, or generated per-subject predictions.
