# Release notes template — v1.0.0

This is the first public software release accompanying the article:

> Rehman et al., “Chin electromyography-based explainable machine learning framework for obstructive sleep apnea severity assessment,” Expert Systems with Applications, 2026. DOI: 10.1016/j.eswa.2026.133654.

## Included

- de-identified MATLAB workflows for ACPN preprocessing/feature extraction;
- canonical 41-feature extraction;
- subject-wise split and LOSO model evaluation;
- subject-level SHAP analysis;
- 1D CNN–BiLSTM baseline;
- UCDDB extraction example;
- documentation and tests.

## Not included

- ACPN clinical recordings;
- ACPN-derived feature matrices;
- real subject identifiers or psychiatric-comorbidity labels;
- third-party dependency source code.
