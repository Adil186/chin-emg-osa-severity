# Contributing

Bug reports and reproducibility feedback are welcome through GitHub Issues.

Do not attach clinical recordings, subject-level feature matrices, real subject identifiers, or protected metadata to an issue or pull request. Use synthetic data to demonstrate a problem.

Before proposing a code change:

1. run `tests/run_all_tests.m`;
2. preserve the canonical 41-feature order;
3. document any change that could affect numerical results; and
4. update `CHANGELOG.md` when appropriate.
